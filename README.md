# Plagiarism Detection System using NLP

This project detects plagiarism between documents using:

- Text Preprocessing
- Bag of Words (BoW)
- Cosine Similarity
- Python, NLTK, Scikit-learn

## Features

- Clean and preprocess text
- Convert text into numerical form
- Calculate similarity between documents
- Detect possible plagiarism

## Technologies Used

- Python
- Jupyter Notebook
- NLTK
- Scikit-learn

## Example Output

Possible plagiarism detected
